# Boundary

OMNIAMIND follows the ecosystem boundary:

    measurement != inference != decision

For this repository, the operational version is:

    orchestration != consciousness != decision

---

## Meaning

Orchestration means:

    organize a structural cognition workflow.

Measurement means:

    produce or request a structural signal under declared conditions.

Inference means:

    interpret the signal.

Decision means:

    act on the interpretation.

OMNIAMIND organizes workflows.

It does not own measurement truth.

It does not infer final meaning.

It does not decide.

---

## Allowed claims

OMNIAMIND may claim:

- a workflow was organized;
- a case was routed to a measurement layer;
- a boundary was preserved;
- an artifact should be validated;
- a stop / continue condition requires inspection;
- external decision is required.

---

## Disallowed claims

OMNIAMIND should not claim by itself:

- this is conscious;
- this is semantically true;
- this is semantically false;
- this is physically proven;
- this is safe;
- this is insecure;
- this downstream decision is correct.

---

## Rule

The output must preserve the distinction:

    workflow organization
    structural measurement
    boundary signal
    interpretation
    external decision
