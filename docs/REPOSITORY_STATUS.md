# OMNIAMIND — Repository Status

## Current position

OMNIAMIND is a structural cognition orchestration layer inside the MB-X.01 / L.O.N. ecosystem.

It is not OMNIA.

It is not consciousness.

It is not artificial consciousness.

It is not a truth oracle.

It is not a semantic judge.

It is not a decision engine.

## Operational role

OMNIAMIND organizes analytical cognition around structural measurement.

OMNIA measures structural behavior.

OMNIA-LIMIT defines stop / continue / retry / escalate boundaries.

The final decision remains external.

## Boundary

Plain boundary:

    measurement != inference != decision

Operational boundary:

    organization != measurement != decision
